NODDO - Preguntas FrecuentesSobre NODDO¿Qué es NODDO?NODDO es una plataforma SaaS que permite a desarrolladores inmobiliarios crear microsites premium para mostrar y vender sus proyectos. Generamos sitios web tipo sala de ventas digital completamente personalizados.¿Cuánto cuesta NODDO?Ofrecemos 3 planes:Proyecto: Para desarrolladores con 1 proyecto ($XX/mes)Studio: Para estudios con múltiples proyectos ($XX/mes)Enterprise: Para grandes desarrolladoras con necesidades personalizadas (precio custom)¿Necesito conocimientos técnicos?No. NODDO es 100% visual y fácil de usar. Si sabes usar Word, puedes usar NODDO.¿Cuánto tarda crear un microsite?Con nuestro sistema de IA, puedes tener tu primer microsite publicado en menos de 30 minutos.---Cuenta y Acceso¿Cómo creo una cuenta?Ve a noddo.ioClick en "Crear cuenta"Ingresa email y contraseñaVerifica tu email¡Listo! Ya puedes crear tu primer proyecto¿Puedo invitar a mi equipo?Sí. En el plan Studio y Enterprise puedes invitar hasta 3 colaboradores. Ve a Dashboard → Equipo → Invitar colaborador.¿Cómo cambio mi contraseña?Dashboard → Cuenta → Cambiar contraseña¿Puedo cambiar de plan?Sí, en cualquier momento. Ve a Dashboard → Cuenta → Plan → Cambiar plan.---Proyectos¿Cómo creo un proyecto?Dashboard → Nuevo ProyectoCompleta la información básica (nombre, ubicación, descripción)Sube imágenes y contenidoPublica cuando estés listo¿Puedo tener múltiples proyectos?Plan Proyecto: 1 proyectoPlan Studio: Hasta 10 proyectosPlan Enterprise: Ilimitados¿Cómo duplico un proyecto?Dashboard → Proyectos → (menu del proyecto) → Clonar proyecto¿Cómo elimino un proyecto?Dashboard → Proyectos → (menu del proyecto) → EliminarCUIDADO: Esta acción no se puede deshacer.¿Qué es el estado "Borrador" vs "Publicado"?Borrador: Solo tú lo ves, puedes editar librementePublicado: Visible públicamente en tu dominio personalizado---Editor - General¿Cómo funciona el autoguardado?NODDO guarda automáticamente cada cambio. No necesitas hacer click en "Guardar".¿Puedo deshacer cambios?Sí, usa Ctrl+Z (Windows) o Cmd+Z (Mac). También puedes revertir a versiones anteriores desde Configuración → Historial.¿Qué navegadores son compatibles?Chrome, Firefox, Safari, Edge (últimas 2 versiones). Recomendamos Chrome para mejor experiencia.¿Funciona en móvil?El editor está optimizado para desktop. Los microsites públicos SÍ son 100% responsive para móviles.---Torres y Tipologías¿Qué es una Torre?Una Torre es un edificio o bloque dentro de tu proyecto. Ejemplo: "Torre A", "Torre Norte", "Bloque 1".¿Qué es una Tipología?Una Tipología es un tipo de unidad o apartamento. Ejemplo: "Apartamento 2 Habitaciones", "Penthouse", "Casa 3 Pisos".¿Cómo creo una Torre?Editor → Torres → Agregar Torre → Completa información (nombre, pisos, unidades)¿Cómo creo una Tipología?Editor → Tipologías → Agregar Tipología → Completa specs (área, habitaciones, baños, precio)¿Puedo subir renders de tipologías?Sí. En cada tipología puedes subir múltiples renders/imágenes que se mostrarán en un slider.¿Qué formato deben tener las imágenes?JPG o PNG, máximo 10MB por imagen. Recomendamos 1920x1080px o superior para mejor calidad.---Inventario¿Cómo funciona el inventario?El inventario te permite marcar el estado de cada unidad:Disponible: Para ventaReservada: Pre-vendida, no disponibleVendida: Ya vendida, no disponible¿Cómo marco una unidad como vendida?Editor → Inventario → Click en la unidad → Cambiar estado a "Vendida"¿Puedo importar inventario desde Excel?Sí. Editor → Inventario → Importar → Sube tu archivo Excel (descarga template primero).¿Los clientes ven las unidades vendidas?Sí, pero marcadas claramente como "Vendida" para generar urgencia.---Galería y Videos¿Cuántas imágenes puedo subir?Ilimitadas en todos los planes.¿Puedo organizar imágenes en categorías?Sí. Editor → Galería → Crear Categoría (ejemplo: "Exteriores", "Interiores", "Amenidades")¿Cómo subo videos?Editor → Videos → Agregar Video → Pega el link de YouTube o Vimeo¿Puedo subir videos directamente?Por ahora solo soportamos YouTube y Vimeo. Para videos privados, sube a YouTube como "No listado".---Ubicación y Mapas¿Cómo funciona el mapa?Usamos Mapbox con vista satelital. Puedes marcar:Ubicación exacta del proyectoPuntos de interés cercanos (colegios, centros comerciales, etc.)¿Cómo agrego puntos de interés?Editor → Ubicación → Agregar POI → Completa info (nombre, tipo, distancia, tiempo)¿Puedo calcular distancias automáticamente?Sí. Al agregar un POI, la distancia y tiempo de viaje se calculan automáticamente desde tu proyecto.---Dominio y Publicación¿Qué dominio recibo?Cada proyecto recibe un subdominio personalizado: `tu-proyecto.noddo.io`¿Puedo usar mi propio dominio?Sí, en plan Studio y Enterprise. Ejemplo: `proyecto.tuempresa.com`¿Cómo conecto mi dominio?Editor → Configuración → Dominio → Sigue las instrucciones (agregar DNS records)¿Cuánto tarda en activarse mi dominio?Entre 24-48 horas después de configurar los DNS records.¿Cómo publico mi proyecto?Editor → Configuración → Publicar → Verificar checklist → Click "Publicar"¿Puedo despublicar un proyecto?Sí. Editor → Configuración → Despublicar. El sitio dejará de ser accesible públicamente.---Leads y Contactos¿Cómo recibo leads?Los visitantes completan el formulario de contacto en tu microsite. Los leads llegan a:Dashboard → Leads (ver todos)Email (notificación inmediata)¿Puedo exportar leads?Sí. Dashboard → Leads → Exportar a Excel¿Se integra con CRM?Sí, con GoHighLevel, HubSpot, Salesforce (según plan). Configurar en Dashboard → Integraciones.¿Los leads incluyen UTM tracking?Sí. Capturamos automáticamente de dónde vino cada lead (Facebook Ads, Google, etc.)---Recursos y Brochures¿Puedo subir PDFs?Sí. Editor → Recursos → Subir Brochure/PDF¿Qué tamaño máximo tienen los PDFs?50MB por archivo.¿Los visitantes pueden descargar el brochure?Sí. Se mostrará un botón de descarga en el microsite.---Tours 360¿Puedo agregar tours virtuales?Sí. Editor → Tour 360 → Subir tour de Matterport u otro proveedor.¿Qué plataformas de tours soportan?Matterport, Kuula, 360Cities, y cualquier iframe embed.¿Puedo subir mi propio tour 360?Sí, si tienes el código embed/iframe.---Soporte y Ayuda¿Cómo contacto soporte?Chat en vivo: Widget en noddo.io (respuesta en minutos)Email: hola@noddo.ioWhatsApp: [tu número]¿Cuál es el horario de soporte?Lunes a Viernes, 9am - 6pm (hora de Colombia)Chat y email: respuesta en <2 horas en horario laboral¿Ofrecen capacitación?Sí. Agenda una demo personalizada: hola@noddo.io¿Hay documentación detallada?Sí. Dashboard → Ayuda (guías completas paso a paso)---Facturación y Pagos¿Cómo pago?Tarjeta de crédito/débito vía Stripe (seguro y encriptado)¿Puedo pagar anualmente?Sí. Planes anuales tienen 20% descuento. Dashboard → Cuenta → Cambiar a anual.¿Puedo cancelar en cualquier momento?Sí. No hay contratos ni permanencia mínima.¿Qué pasa si cancelo?Tus proyectos se despublicanConservas acceso hasta fin del período pagadoPuedes exportar tus datos antes de cancelar¿Ofrecen reembolsos?Sí, garantía de satisfacción de 14 días en nuevas suscripciones.---Seguridad y Privacidad¿Mis datos están seguros?Sí. Usamos encriptación SSL, hosting en servidores certificados, backups diarios automáticos.¿Quién ve mi información?Solo tú y tu equipo. NUNCA compartimos datos con terceros.¿Cumplen con GDPR?Sí. Lee nuestra Política de Privacidad: noddo.io/legal/privacidad¿Hacen backups?Sí. Backups automáticos diarios. Puedes restaurar versiones anteriores desde Configuración → Historial.---Problemas ComunesNo puedo subir imágenesVerifica que el archivo sea JPG o PNG (no GIF, no WEBP)Verifica que pese menos de 10MBIntenta con otro navegadorSi persiste: contacta soporteEl mapa no cargaVerifica tu conexión a internetRecarga la página (Ctrl+R o Cmd+R)Limpia cache del navegadorSi persiste: contacta soporteLos cambios no se guardanVerifica que tienes conexión a internetEl autoguardado toma 2-3 segundosBusca el ícono "✓ Guardado" en la esquina superiorSi ves "Error al guardar": contacta soporteOlvidé mi contraseñaDashboard → Login → "¿Olvidaste tu contraseña?" → Recibirás email para resetearNo recibo el email de verificaciónRevisa carpeta de spam/promocionesEspera 5 minutos (puede demorar)Solicita reenvío desde el loginSi no llega: contacta hola@noddo.io---IA y Automatización¿Qué hace la IA de NODDO?La IA te ayuda a:Crear proyectos desde archivos Excel (parser automático)Generar puntos de interés cercanos automáticamenteSugerir textos de marketingOptimizar descripciones SEO¿Cómo uso el parser de unidades?Editor → Inventario → Importar → Sube Excel → La IA detecta columnas automáticamente¿Puedo editar lo que sugiere la IA?Sí, siempre. La IA solo sugiere, tú decides qué usar.---Integraciones¿Se integra con Meta Ads?Sí. Tracking automático de conversiones con Meta Pixel.¿Se integra con Google Analytics?Sí. Dashboard → Integraciones → Google Analytics → Conectar¿Se integra con WhatsApp?Sí. Cada microsite incluye botón flotante de WhatsApp configurable.¿Puedo conectar mi CRM?Sí. Soportamos GoHighLevel, HubSpot, Salesforce (según plan).
